# Sistem Informasi Bahan Ajar dan Tracking Delivery Order

Aplikasi web sederhana berbasis Vue.js yang digunakan untuk mengelola data stok bahan ajar serta melakukan monitoring Delivery Order (DO) pada proses distribusi bahan ajar.

## Demo Aplikasi

https://reetempest00.github.io/finalStock-DO/

## Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript (ES6)
* Vue.js 2
* LocalStorage

## Fitur Utama

### Manajemen Stok Bahan Ajar

* Menampilkan data stok bahan ajar
* Menambah data bahan ajar
* Mengubah data bahan ajar
* Menghapus data bahan ajar
* Filter berdasarkan UPBJJ
* Filter berdasarkan kategori
* Filter stok bermasalah
* Sorting data stok
* Penyimpanan data menggunakan LocalStorage

### Tracking Delivery Order

* Menampilkan data Delivery Order (DO)
* Pencarian berdasarkan nomor DO
* Filter berdasarkan status pengiriman
* Menampilkan statistik pengiriman
* Menampilkan detail perjalanan paket
* Menambah Delivery Order baru
* Menghapus Delivery Order
* Penyimpanan data menggunakan LocalStorage
